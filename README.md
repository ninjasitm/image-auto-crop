# NITM Image Auto-Crop

A collection of command-line tools for automatically cropping and processing SVG images. Particularly useful for cleaning up images generated by AI image generators.

## Project Structure

```
nitm-image-auto-crop/
├── svg-tools/          # SVG processing utilities
│   ├── svg_auto_crop.py    # Main SVG cropping and processing tool
│   └── README.md           # Detailed documentation for svg-auto-crop
├── LICENSE             # MIT License
└── README.md           # This file
```

## Available Tools

### svg-auto-crop (`svg-tools/svg_auto_crop.py`)

Analyze, crop, and process SVG files by computing tight bounding boxes and removing whitespace. Supports batch processing, verification, responsive mode, CSS class assignment, and PNG rasterization.

For complete documentation, usage examples, and all available options, see [svg-tools/README.md](svg-tools/README.md).

## Installation

Clone the repository:

```bash
git clone https://github.com/ninjasitm/nitm-image-auto-crop.git
cd nitm-image-auto-crop
```

Optional: Install cairosvg for PNG rasterization support:

```bash
pip install cairosvg
```
